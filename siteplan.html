W05 Project — Cas Perche

Site Name

Name: Cas Perche Creative — Makeup Artistry

Why this name? It clearly reflects my brand (Cas Perche) and the focus on creative / editorial makeup as art. It's memorable, short, and flexible for a portfolio and learning resources.

Optional: Domain ideas (availability not required)
  • casperchecreative.com
  • casmakeupart.art
  • yirutlovebeauty.com

Domain purchase is not required for this class; these are just ideas.

Site Purpose

This website is a small, elegant hub that showcases creative and editorial makeup looks, provides concise how‑to tips and tools, and offers contact links for collaborations. It will be responsive, accessible, and a foundation to practice modern HTML, CSS, and JavaScript (events, filtering, dynamic content).

  • Showcase a gallery of looks with short descriptions.
  • Offer simple tutorials/resources (brushes, products, techniques).
  • Share an “About” story and links to social media.

Scenarios

  • Q: Where can I see Cas’s editorial looks and product details?
    A: On the Gallery page—each card lists the look, products, and a short note.
  • Q: I’m a brand—how do I collaborate?
    A: Visit the Contact page for email/social links and a simple inquiry form.

CSS Notes

  • Mobile‑first CSS with a desktop breakpoint at 48rem.
  • Use CSS variables for color schema to keep things consistent.
  • Accessible focus states and readable line length.

Testing & Self‑Check

  • Validate HTML/CSS; check Console for JS errors.
  • Run Lighthouse (mobile/desktop): Accessibility, Best Practices, SEO.
  • Check color contrast and keyboard navigation.
